    Для обработки ошибок импортируем метод по работе с этими исключениями, из Flask:
    
        from flask import abort
    

    Пример использования
    
        abort(404)
    

    Можно как обычно перенаправлять пользователя  при помощи метода redirect
    
        redirect(url_for('login'))
    

    Но метод abort возвращает ошибку с указанным номером, и автоматически редиректит
    клиента на стандартную страницу ошибок, которую если надо, то можно переопределить.

    Переопределение страницы ошибок, обычно мы используем метод декоратор @app.route("/errors")
    Если мы работаем со стр ошибок, то используем  @app.errorhandler(404)
    
        @app.errorhandler(404)
        def page_not_found(error):
            return render_template('page_not_found.html'), 404
